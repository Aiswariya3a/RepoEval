# Project Guide

## Project Context

**Project:** Project Intelligence & Evaluation Platform
**Phase:** 1 of 9 — Foundation & Authentication

An AI-powered platform that automatically analyzes GitHub repositories and generates comprehensive evaluation reports using static code analysis, repository mining, and AI agents.

## Core Value

Automatically generate objective, rubric-based evaluation reports for any GitHub repository within minutes.
