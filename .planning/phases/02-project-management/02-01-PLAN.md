       - Router: `router = APIRouter(prefix="/api/projects", tags=["projects"])`
       - Imports: `from fastapi import APIRouter, Depends, HTTPException, status`, `from sqlalchemy import select, func`, `from sqlalchemy.ext.asyncio import AsyncSession`, `from app.database import get_db`, `from app.auth.dependencies import get_current_user`, `from app.models.project import Project`, `from app.models.user import User`, `from app.projects.schemas import ProjectCreate, ProjectUpdate, ProjectResponse, ProjectListResponse`, `import uuid`
       - Pagination defaults: page=1 (query param), page_size=12 (query param, max 100)

       Endpoints (per D-25, D-26, D-27):

       a) `GET /api/projects` — List user's projects (paginated, newest first per D-06)
          - Auth: `user: User = Depends(get_current_user)`
          - Query params: `page: int = Query(1, ge=1)`, `page_size: int = Query(12, ge=1, le=100)`
          - Count query: `select(func.count(Project.id)).where(Project.owner_id == user.id)`
          - Data query: `select(Project).where(Project.owner_id == user.id).order_by(Project.created_at.desc()).offset(offset).limit(page_size)`
          - Compute total_pages = ceil(total / page_size)
          - Return ProjectListResponse

       b) `POST /api/projects` — Create project (per D-16, D-29)
          - Auth: same
          - Body: ProjectCreate
          - Create Project ORM instance with owner_id=user.id
          - Return 201 with ProjectResponse

       c) `GET /api/projects/{project_id}` — Get single project detail
          - Auth: same
          - Query: `select(Project).where(Project.id == project_id, Project.owner_id == user.id)`
          - If not found: 404
          - Return ProjectResponse

       d) `PATCH /api/projects/{project_id}` — Update project (per D-16)
          - Auth: same
          - Body: ProjectUpdate
          - Only update fields that are not None (exclude unset fields)
          - Update `updated_at` to current UTC time
          - If not found: 404
          - Return ProjectResponse

       e) `DELETE /api/projects/{project_id}` — Delete project (per D-22)
          - Auth: same
          - Query and delete with owner_id filter
          - If not found: 404
          - Return 204 No Content

       Error handling: Use `raise HTTPException(status_code=404, detail="Project not found")` for missing projects. Auth failures handled by get_current_user (raises 401).
